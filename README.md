git # BehavioralGeneticsSQL
人类行为遗传数据库，创建于2015-09-26

# 使用技术
+ AngularJS
+ BootStrap
+ JQuery

# 后台和数据库
+ loopBack
+ mySQL

# 主要功能
+ Phenotype
+ Gene/Region
+ Marker
+ Pathway
+ Browser

# 安装运行

安装依赖：
```
cd <your project folder>
npm install
```
然后运行它：
```
slc run
```
如果需要使用loopback构建的`/explorer`和`/api`，你可以使用
```
node .
```
然后在浏览器打开 [http://localhost:9000/](http://localhost:9000/)
